# Recursion

## Essential parts

- Base Case
- Different Input

## Where things go wrong

- No base case
- Forgetting to return or returning the wrong thing
- Stack overflow

## Pure recursion tips

- For arrays
  - `slice`, `spread operator` and `concat`
  - Do not mutate original arrays
- Strings
  - Are immutable
  - `slice`, `substr` or `substring`
- Objects
  - `Object.assign` or the `spread operator`
