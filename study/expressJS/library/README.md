# Project Steps

#### $ `npm init`

#### Create file `<projectFolder>/.npmrc` with:

#### `save=true`

#### `save-exact=true`

#### $ `npm install express chalk debug morgan`

#### $ `npm install jquery bootstrap`

#### $ `npm install ejs`

#### Install vscode EJS extension

#

#### ALL: $ `env DEBUG="*" node app.js`

#### APP only: $ `env DEBUG="app" node app.js`
